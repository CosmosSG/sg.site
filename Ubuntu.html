VSCode

Install

  1. インストール用のdebファイルをVisual Studio Code 公式サイトからダンロードする

  2. ターミナルを開いてアップデートを行う

    sudo apt update

  3. ターミナルを開いてダウンロード先のディレクトリを開く

    cd ~/Downloads

  4. 実行権限を与える

    chmod +x ./code_1.80.2-1690491597_amd64.deb

  5. root権限でインストールを行う

    sudo apt install ./code_1.80.2-1690491597_amd64.deb

Anaconda

Install

  1. インストール用のshファイルをAnnaconda公式サイトからダウンロードする

  2. ターミナルを開いてダウンロード先のディレクトリを開く

    cd ~/Downloads

  3. bashで実行

    bash ./Anaconda3-2023.07-1-Linux-x86_64.sh

  4. インストールが始まると以下のようにライセンス規約に同意するか聞かれるため、同意する場合は「yes」と入力する

    ==================================================
End User License Agreement - Anaconda Distribution
==================================================

Copyright 2015-2023, Anaconda, Inc.

All rights reserved under the 3-clause BSD License:

This End User License Agreement (the "Agreement") is a legal agreement between y
ou and Anaconda, Inc. ("Anaconda") and governs your use of Anaconda Distribution
(which was formerly known as Anaconda Individual Edition).

･･･

Do you accept the license terms? [yes|no]
[no] >>> yes

  5. 以下のようにインストールする場所を聞かれるため、特に理由がなければEnterを押してホームディレクトリにインストールする

    Anaconda3 will now be installed into this location:
/home/[UserName]/anaconda3

  6. ダウンロードが終了すると以下のようにPATHを通すか聞かれるため、「yes」と入力する

    Do you wish the installer to initialize Anaconda3
by running conda init? [yes|no]
[no] >>> yes

  7. 一度ターミナルを開き直してバージョンを確認する

    conda -V

    コマンドが見つからないと表示された場合は以下のコードをホームディレクトリにある.bashrcファイルに追記する

    # >>> conda initialize >>>
# !! Contents within this block are managed by 'conda init' !!
__conda_setup="$('/home/haru/anaconda3/bin/conda' 'shell.bash' 'hook' 2> /dev/null)"
if [ $? -eq 0 ]; then
    eval "$__conda_setup"
else
    if [ -f "/home/haru/anaconda3/etc/profile.d/conda.sh" ]; then
        . "/home/haru/anaconda3/etc/profile.d/conda.sh"
    else
        export PATH="/home/haru/anaconda3/bin:$PATH"
    fi
fi
unset __conda_setup
# <<< conda initialize <<<

    記述できたら.bashrcを更新する

    source ~/.bashrc

Anaconda配下のパッケージの更新

パッケージの更新

conda upgrade --all

インストールが終わったら古いパッケージを削除する

conda clean -packages

pipの更新

conda update pip
